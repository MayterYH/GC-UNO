# ISEE Database for Nonlinear Force-Free Field of Solar Active Regions

This directory contains the dataset used for the real-time coronal magnetic field extrapolation model. The data is sourced from the **ISEE NLFFF Database**.

**Official Website:** [https://hinode.isee.nagoya-u.ac.jp/nlfff_database/](https://hinode.isee.nagoya-u.ac.jp/nlfff_database/)

## 📝 Data Description

The database contains three-dimensional (3D) magnetic fields of solar active regions analyzed by Kusano *et al.* (2020). The 3D magnetic fields are extrapolated using the magnetohydrodynamic (MHD) relaxation method (Inoue *et al.*, 2013) from vector magnetic field data observed by the Solar Dynamics Observatory (SDO/HMI). 

Specifically, this dataset utilizes Space weather HMI Active Region Patch (SHARP) data that has been remapped to a Lambert Cylindrical Equal-Area (CEA) projection. For a detailed list and parameters of the sampled data, please refer to Kusano *et al.* (2020).
